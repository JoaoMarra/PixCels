# Draw your sprites:

https://joaomarra.github.io/PixCels/

## What is PixCels

Pixcels is a simple tool to draw pixel arts. Select your color, your pixel size, click around and do your drawings. Next, save it to a PNG file in the size you want (you draw on a big canvas and export the size you need). It is always a square...

## Adding colors
You can add colors using "New color" button. Add more than one HEX color separting using space.

## Hot-keys

### A - S
Decrease/Increase `dot size`.

### Q - W
Change the current color, moving left/right through your pallet.

### E - D
Change the current layer to the next or previous on.

### Arrows
Move the layer based on the `dot size`.

### C
Color the square, same as click of the mouse/touch

### HOLD Shift
Erase the dot instead of coloring it when click/touch or use `C`. It will also show the dots boundaries.
